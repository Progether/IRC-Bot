IRC-Bot
=======
progether IRC bot


Usage
-----------

**Mail**

To send a mail message:

   !!mail <user> <message>

To view messages:

   !!mymail

To del message:

   !!delmail <id>

All commands can also be performed as private messages like this:

   /msg progether !!mymail

All interactions are encouraged to be done through private messages.
